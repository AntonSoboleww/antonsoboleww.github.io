Проекты

Парсер ВК

Программа с интерфейсом, служащая для создания полного дампа страницы ВК по её токену с помощью API. После полного парсинга страницы, создаётся структура HTML документов, которая позволяет локально просмотреть все данные на странице, которые были в момент создания её копии.


Прототип Социальной Сети

Небольшой сайт с возможностью регистрации, публикации контента, выставлением никнеймов, чтением глобальной ленты, а так же имеет в себе API. Создан исключительно для практики знаний в фрэймворке Django.
